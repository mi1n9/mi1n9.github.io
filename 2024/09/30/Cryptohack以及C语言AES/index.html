AES128ECB的C语言实现

C语言实现AES跟python还是蛮多不一样的，借着这次机会也算是复习一下C的一些知识。首先需要实现的函数还是那几个，添加轮密钥，行移位，列混淆，字节代换，密钥扩展……

首先需要定义一下密钥结构体以及密钥扩展相关功能的函数定义。

  1. 在AES-128中，每个轮密钥由4个uint32_t类型的词组成，即4个32位的字。这意味着每个轮密钥占用16字节。
  2. 总轮密钥数量：既然每个轮密钥由4个uint32_t组成，而AES-128需要11个轮密钥，那么总共需要的uint32_t数量就是11轮乘以每轮的4个词，即11×4=44。

因此，对于AES-128：

  • eK[44]：用于存储加密过程中使用的44个uint32_t轮密钥。
  • dK[44]：用于存储解密过程中使用的44个uint32_t轮密钥。

  1. BLOCKSIZE:

    定义了AES算法中数据块的大小为16字节（128位）。

  2. LOAD32H:

    这个函数用于将4个字节的数据加载到一个uint32_t类型的变量中。它按照大端字节序（最高位字节在前）将字节组合成一个32位的整数。

  3. STORE32H:

    这个函数用于将一个uint32_t类型的变量拆分成4个字节，并存储到一个字节数组中。它同样按照大端字节序操作。

  4. BYTE:

    这个函数用于从uint32_t类型的变量中提取指定位置的字节。参数n表示要提取的字节的位置（从0开始）。

  5. MIX:

    这个函数用于在密钥扩展中进行字节替换和循环左移操作。它首先使用S盒（S数组）对字节进行替换，然后将结果循环左移一位

  6. ROL32:

    这个函数用于将uint32_t类型的变量循环左移指定的位数。循环左移意味着在移位过程中，移出的位会在另一边重新进入。

  7. ROR32:

    这个函数用于将uint32_t类型的变量循环右移指定的位数。循环右移与循环左移相反，移出的位会从另一边重新进入。

伽罗华域上的乘法

伽罗华域一般用 GF(2^M^)表示，这个域中含有 2^M^个元素,GF(2^M^)上的四则运算是基于多项式运算的，一般都是这种结构f ( x ) = x ^6^ + x ^4^ + x^2^ + x + 1 。AES加解密算法中，使用的不可约多项式（irreducible polynomial）为P ( x ) = x^8^ + x^4^ + x^3^ +x+1 .

加法

在伽罗华域中，加法是模2运算，也就是忽略进位的加法，等同于计算机中的XOR异或，即 1^1=0, 1^0=1, 0^0=0。

乘法

伽罗华域中的乘法是基于多项式运算的，比如：5 = 00000101b = ( 2^2^ + 1 ) ，对应多项式为 ( x^2^ + 1 ) .

3∗7=(x+1)(x^2^+x+1)=x^3^+x^2^+x+*x^2^+x+1=x^3^+(x^2^+x^2^)+(x+x)+1=*x^3^+1.
